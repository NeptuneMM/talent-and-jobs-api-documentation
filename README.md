
# Talent&Jobs API documentation

This is the documentation for talent&jobs API.



## API Reference

#### Get all items

```http
  GET /api/jobswithfeatures
```

| Parameter | Type     | Description                | Status | Auth Type |
| :-------- | :------- | :------------------------- |:-------| :-------- |
| `none` | `Object` | This will return both features and nonfeatures jobs | 200 | No Auth is required
